# draggable-demo
Simple editable sections where you can add sections and edit texts and its position with text alignment, made with Vue
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

# Dependencies
- Vuedraggable
- Bootstrap-vue
